# Practice Assessment

> **Important context for AI-103**: As of April 2026 there is no separate "AI-103" Microsoft certification — the **AI-102: Designing and Implementing a Microsoft Azure AI Solution** certification is the official credential, and the AI-103 study guide here is an extended, agent-and-Foundry-heavy companion built on top of the AI-102 skills measured. **Use the AI-102 practice assessment** — it is the same skill domains.

Microsoft offers a **free official practice assessment** for AI-102. It is designed by the same teams that build the real exam, so question style, scenario depth, and difficulty are the closest possible match to what you will see on test day.

## How it works

1. Sign in with your **Microsoft Learn account** (the same one tied to your certification profile).
2. Take the assessment as many times as you want — questions and answer order shuffle each attempt.
3. After every question you get a detailed explanation **and** a link to the exact Microsoft Learn module that covers the concept, so you can close knowledge gaps immediately.
4. At the end you get a per-skill-area score breakdown that mirrors the real exam's "Skills measured" outline.

> The assessment is not graded against a passing score the way the real exam is — treat 80%+ on each skill area as a healthy readiness signal.

## Suggested study loop

```text
Read this guide  →  Take AI-102 practice assessment  →  Review weak skill area on Learn  →  Retake assessment  →  Schedule real exam
```

Aim for **two clean passes at 85%+** before booking the real exam. Pay extra attention to the **agent / Foundry** topics covered in this guide — they appear on the modern AI-102 question pool and the practice assessment refresh.
